# Ejecutar en Desarrollo

1. Clonar el repositorio.
2. Ejecutar el siguiente comando:

    ```bash 
    yarn install
    ```
3. Tener Nest CLI instalado globalmente:

    ```bash
    npm i -g @nestjs/cli
    ```1

4. Levantar la base de datos con Docker:

    ```bash
    docker-compose up -d
    ```


## Stack usado
*MongoDB
*Nest
